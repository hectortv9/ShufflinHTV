YouTube Chromecast Speed Hack

Casting YouTube videos disables some features (e.g. playback speed). Clicking the extension's camera button embeds the video you are watching into this page so you can follow the steps below and adjust the playback speed while casting.

  1. Cast this page (click the cast button in the Chrome Toolbar).
  2. Copy and paste a YouTube URL below (skip this step if the video was already embedded by the extension.
  3. Play the video, set the playback speed.
  4. Click the fullscreen icon in the bottom right of the video.

You can also create a bookmarklet (bookmark stored in a web browser that contains JavaScript commands to add functionality to the browser as a "one-click" tool).

  1. Create a bookmark with the following code as URL:
    javascript:(function () { (window.location.href = "https://greggman.github.io/youtube_chromecast_speed_hack/?" + window.location.href) }());
  2. Open the bookmark when you're on a YouTube page to quickly apply this speed hack.
